## üöÄ Getting Started

### Installation
Download the latest APK from the **[Releases](https://github.com/DarkKnight2122/Audiobookplayer/releases)** section. 
*Note: For the best experience (including real-time blur), Android 12+ is recommended.*

### Usage
1. Open the app and grant storage permissions.
2. Place your audiobook folders in your phone's `/Audiobooks`, `/Music`, or `/Download` folder.
3. The app will automatically scan and list your books.
4. Tap a book to start listening.
